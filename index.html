LDC Broad Phonetic Class Speech Activity Detector

ldc-bpcsad is a speech activity detector (SAD) developed at Linguistic Data Consortium (LDC) based on recognition of broad phonetic classes. The tool was originally developed circa 2012 for internal use and now is being made available to the public. It can be accessed through either the ldc-bpcsad command line tool or directly through an API.

Installation

To install ldc-bpcsad and its dependencies, please follow the detailed instructions in the installation guide.
